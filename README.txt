#Pokedex Web App (Bootcamp DIO - Santander):

Uma aplicação web simples que exibe uma lista de Pokémon usando a API PokeAPI. Os Pokémon são exibidos em uma grade com informações básicas e tipos correspondentes. Os usuários podem carregar mais Pokémon usando o botão "Load More".

#Tecnologias Utilizadas:

- HTML
- CSS
- JavaScript

#Funcionalidades:

Exibe uma lista de Pokémon com informações básicas. 

Os Pokémon são categorizados por tipo e têm cores de fundo correspondentes.

Implementação de paginação com botão "Load More" para carregar mais Pokémon.

Interface responsiva para diferentes tamanhos de tela.

#Como Usar: 

Clone ou baixe este repositório para a sua máquina local.

Abra o arquivo index.html em um navegador da web.

#Estrutura do Projeto:

index.html: Página principal da aplicação, exibe a lista de Pokémon e contém o HTML estrutural.

assets/css/global.css: Estilos globais para a aplicação.

assets/css/pokedex.css: Estilos específicos para a exibição dos Pokémon.

assets/js/main.js: Script JavaScript responsável por carregar os Pokémon da API e manipular a página.
